### Git Operations
```
$ git remote add origin  <REMOTE_URL> 
$ git remote add origin https://github.com/hustbill/tech-blog.git

# Sets the new remote
$ git remote -v
# Verifies the new remote URL

git push --set-upstream origin master

```
